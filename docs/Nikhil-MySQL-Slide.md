
MySQL Database Integration
Built Python FastAPI service to replace hardcoded products with MySQL database
Implemented database schema with products table (id, name, category, price, stock)
Created RESTful API endpoints for product retrieval and health checks
Integrated ProductService in Angular microfrontend with HttpClient
Established complete data flow from MySQL to UI via FastAPI
Dockerized MySQL and API services with health checks
Used raw SQL queries with pymysql for lightweight database operations


Key Outcomes
Dynamic product catalog powered by relational database
Seamless integration between Angular frontend and Python backend
Exposed /metrics endpoint for Prometheus monitoring integration
Database seeding capability for easy data management
Scalable microservices architecture with proper separation of concerns
API documentation available at /docs endpoint
Foundation for future CRUD operations and data persistence
